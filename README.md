# Akshat Jain — Portfolio Website

My personal portfolio website, built to showcase my projects, skills, and certifications in Cyber Security and Data Science.


## Features

- Responsive design with a dark mode toggle
- Dedicated case-study page for every project with overview, tech stack, and screenshots
- Skills section organized by category (Programming, Cyber Security, Data Science & ML, Web Development, Tools)
- Certifications section with direct links to certificates
- Working contact form for recruiters and collaborators to reach out

## Tech Stack

- HTML5, CSS3, JavaScript
- Bootstrap 5
- Bootstrap Icons / Font Awesome
- Formspree (contact form handling)

## Project Structure

```
├── index3.html          # Main landing page
├── project-*.html       # Individual project case-study pages
├── style3.css            # Stylesheet
├── DarkMode.js           # Dark mode toggle logic
├── images/                # Project thumbnails and site images
└── screenshots/           # Project screenshots used on case-study pages
```
